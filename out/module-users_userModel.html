(static) checkNameAvailability(params, callback)

Queries database for a userName. If the userName already exists in the database, the callback will take a valid error. If the userName does not exist then the callback will take an undefined error and the results of the query.
Parameters:
Name Type Description
params string a userName
callback function a callback which will take the arguments err and results from the database query
